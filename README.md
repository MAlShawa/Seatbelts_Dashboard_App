Github Repository for files used in building the **Seatbelts_Dashboard App**, a Shiny application for the *Developing Data Products* course, as part of the *Data Science Specialization* from Johns Hopkins University.

**Seatbelts_Dashboard App and its Code:** 

- Application is available at shinyapps.io: https://masapps.shinyapps.io/Seatbelts_Dashboard/ 
- Documentation and instructions are included and embedded in the application's sidebar and 
main tab panels, and through its "About Data" and "About App" tabs.
- The ui.R and server.R files of the app are available in this repository: https://github.com/MAlShawa/Seatbelts_Dashboard_App

**Seatbelts_Dashboard App's Presentation and its Code:** 

- Slides are available at Rpubs: https://rpubs.com/MAlShawa/1382929
- Note that: R Presenter format, is obsolete, became legacy in 2022, and RStudio replaced it with 
R Markdown ioslides_presentation format. Still produces presentations with HTML 5 format, as R Presenter.
- File for the slides is available in this repository: https://github.com/MAlShawa/Seatbelts_Dashboard_App
